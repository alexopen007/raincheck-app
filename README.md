# 🌧️ RainCheck

**Premium umbrella rental app — never get caught without cover.**

## Tech Stack

- **React Native** with **Expo SDK 54**
- **expo-router** for file-based navigation
- **Zustand** for state management (persisted with AsyncStorage)
- **TypeScript** throughout
- **react-native-maps** for station map
- **@gorhom/bottom-sheet** for bottom sheets
- **expo-camera** for QR code scanning

## Features

- 🗺️ **Interactive Map** — Find nearby umbrella stations with real-time availability
- 🔍 **Station Search** — Filter stations by name
- ❤️ **Favorites** — Save your go-to stations
- 📷 **QR Scan & Rent** — Scan a station QR code to unlock an umbrella
- ✏️ **Manual Code Entry** — 6-digit fallback if QR won't scan
- 🔄 **Return Flow** — Scan any station to return; confirm with cost summary
- ⭐ **Post-Return Rating** — Rate your umbrella experience
- 📊 **Activity History** — View all past and active rentals
- 🧾 **Digital Receipts** — Download or share rental receipts as images
- 🔔 **Notifications** — Rain alerts, rental duration warnings, promos
- 💳 **Wallet** — Balance top-up, add cards/UPI, transaction history
- 🎁 **Referral Program** — Invite friends, earn ₹50 credit each
- ❓ **Help & FAQ** — Searchable FAQ with accordion answers
- 🌦️ **Weather Banner** — Live weather alerts with action prompts
- 🛡️ **Damage Reporting** — Report umbrella issues during return
- 📱 **Onboarding** — 3-slide intro for new users
- 🔐 **Phone Auth** — OTP-based authentication flow

## Getting Started

```bash
# Install dependencies
npm install

# Start the development server
npx expo start
```

Then open in Expo Go on your device, or press `i` for iOS simulator / `a` for Android emulator.
